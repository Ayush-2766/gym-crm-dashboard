# Gym CRM AI

An AI-powered Gym CRM system built using Flask, Google Sheets, Twilio WhatsApp API, and Gemini AI.

## Features

* AI-generated customer replies
* WhatsApp chatbot integration
* Lead scoring (HOT / WARM / COLD)
* Intent detection
* Google Sheets CRM storage
* Flask dashboard
* Real-time webhook handling

## Tech Stack

* Python
* Flask
* Twilio
* Google Sheets API
* Gemini AI
* Ngrok

## How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Add environment variables in `.env`

3. Run Flask app:

```bash
python3 app.py
```

4. Start ngrok:

```bash
ngrok http 5000
```
